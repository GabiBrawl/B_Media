# B_Media Audiophile Gear Recommendations

A user-friendly static website showcasing audiophile gear recommendations from B_Media. This project is not affiliated with B_Media; it's a fan-made resource based on their official Linktree recommendations.

[![pages-build-deployment](https://github.com/GabiBrawl/B_Media/actions/workflows/pages/pages-build-deployment/badge.svg)](https://github.com/GabiBrawl/B_Media/actions/workflows/pages/pages-build-deployment)

## Features

- **Responsive Design**: Works across all devices
- **B_Media Picks**: Highlighted recommendations with special badges
- **Filtering**: Search by name, filter by category, price range, and more!
- **Affiliate Links**: Direct links to purchase products, directly supporting B_Media
- **Product details**: Directly condensing important specs and details of products in the same page

## Filters

- **Search**: Find recommended products by name
- **Category**: Filter by specific gear type
- **Price Range**: Filter by price brackets ($0-50, $50-100, etc.)
- **B_Media Picks**: Show only recommended products

No dependencies or servers required - a static page with pure HTML, CSS, and JavaScript!
